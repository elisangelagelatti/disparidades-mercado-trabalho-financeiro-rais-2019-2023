Esta pasta contém os scripts em R e SQL utilizados para extração, tratamento e análise dos dados da RAIS no período de 2019 a 2023.

## Fonte dos dados

Os dados utilizados no projeto são públicos, provenientes da Relação Anual de Informações Sociais (RAIS), do Ministério do Trabalho e Emprego, e foram acessados por meio do Google BigQuery, utilizando os datasets disponibilizados pela Base dos Dados:

- Base dos Dados – RAIS (BigQuery)
  - `basedosdados.br_me_rais.microdados_vinculos`
  - `basedosdados.br_me_rais.microdados_estabelecimentos`

⚠️ **Os microdados da RAIS não são disponibilizados neste repositório**, em conformidade com as restrições legais e de confidencialidade.
