# Scripts
Esta pasta contém os scripts em R e SQL utilizados para extração, tratamento e análise dos dados da RAIS no período de 2019 a 2023.

⚠️ Os microdados da RAIS não são disponibilizados neste repositório, em conformidade com as restrições legais e de confidencialidade.

## Fonte dos dados
Os dados utilizados no projeto são públicos e foram acessados por meio do Google BigQuery, utilizando o dataset disponibilizado pela Base dos Dados:

- Base dos Dados (BigQuery) – RAIS  
  https://basedosdados.org/dataset/br-me-rais

Os dados também estão disponíveis no portal oficial do Ministério do Trabalho e Emprego:

- Ministério do Trabalho e Emprego – RAIS  
  https://www.gov.br/trabalho-e-emprego/pt-br/assuntos/estatisticas/rais
